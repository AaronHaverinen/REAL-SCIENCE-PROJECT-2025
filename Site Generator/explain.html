Project Management

Importance of Project Management

At the beginning of a project, it might not seem necessary as only a few files are used a code is simple. However, as you get deeper into a project, there are so much more files (I have at least 30) and 100x more code. This makes debugging a pain if you don't manage your project well.

What is Git?

Git is a version control system that tracks changes in computer files, making it easy to manage and collaborate on projects.

How did I use Git to manage our code?

After saving a file, I would be able to commit the file in Sourcetree, then push it to my GitHub page. The allowed for me to be able to easily see old versions of my code and debug it quickly.

Something didn't add up. After another day of researching, I was finally able to detect the error. I needed to stylize an anchor link with an href as a button in css. I went through countless websites, trying to find some css code that would at least point me in the right direction. And finally, I found it. The Dread of the Quiz Button

I don't know if it's just me, or does everyone have trouble with buttons. Well, JavaScript was never my strongest suit, probably because I'd only used it once before this project. Let's get into the story. So I was coding the quiz portion of my website and everything was working perfectly: the questions were showing up, you could select an answer, etc. But there was one thing that stole another hour of my away: the submit button. To this day, I still hate that thing. In my code, I called a variable idx and used it so see if answers were submitted. Then, the submit button code should have looped through it to see how many answers were correct. But it didn't, for some reason. I kept working and working and working, yet nothing seemed to work. I even tried looking at other people's websites to try and see the difference, but there was none.

I sat in that chair, almost pulling my hair out of my scalp from the frustration. But finally, as almost all hope was lost, some knowledge was bestowed upon me. What if I looped through the program while checking the input and query selector? I tried that and in theory, it worked. All the logic aligned and there wasn't any error message in the console. So I tried the button on my website and it worked. The only problem was that no matter how many answers were correct, it still said, "You scored 0 out of 12!"

So image a little kid who studied all the pages and took the quiz, getting every answer correct. His heart would shatter into millions of pieces and those millions of pieces would shatter into millions of pieces each if he found out that he got "0 out of 12". This was quite a large problem. I sat there for another long period of time, trying to debug it. And after a good amount of rereading the code, I found out the problem. I used ' instead of `. That's it?! Just a typo wasting all my time?! I guess that's how coding is...

CSV and Databasing

Why is Google Sheets Helpful?

Instead of writing the descriptions in the coding application and all the terms, a spreadsheet is much more orgasinsed, where you can have designated columns for different things. Additionally, Google Sheets can check grammar mistakes as well and it can autofill information, saving a lot of time.

What is a CSV File and Why does it Matter?

A python script can't read a spread sheet in Google Sheets alone so we need to convert it into a different format. A CSV file, or "comma seperated values", let's the python program read it just like we would a normal Google Sheet. Once it encounters a comma, it stops, signifying that there is a section break.

How Can Google Sheets Create Multiple Websites at Once?

You can download multiple tabs into different CSV files and with those, all you need to do is plug the name into my current python program. This way, html files will all be generated, each completely different and reliant on the information you put in the sheet.

Potential Further Than This Project

This CSV databasing can be sold as a service (SAAS - Software as a Service) where others can generate a website through my service (aka. this program). This has huge potential and can be sold to make a lot of passive income. This also makes website generation for the general public much easier.

Interesting Code Solutions

Map Coordinates

This basically turns the image into a map with coordinates that show an area in which someone can click that will lead them to a certain exhibit. It works with creating a rectagular "box" in which each pixel leads to the same exhibit.

Forward/Back Buttons

Buttons again... But the logic behind these is pretty extraordinary. The previous button will take the ordinal place of the current exhibit and subtract that by one, therefore creating a link to the previous exhibit. The next button will do the opposite, adding one to the ordinal place and linking to the next exhibit. However, the first exhibit cannot go back and the last cannot go forward so we give conditionals that apply to the exhibits that are not first/last respectively.
